One-time setup

To let this page save changes to your bot, paste a GitHub access token. It's stored only in this browser — you do this once.

1. Open github.com/settings/personal-access-tokens/new
2. Token name: tee-time-ui
3. Repository access → Only select repositories → pick golfteetime
4. Repository permissions → set Contents = Read and write, and Actions = Read and write
5. Generate token, copy the github_pat_… string, paste below
